# AI Voice Assistant (MVP)

---

## 🎯 功能范围
- 语音输入
- ChatGPT AI 回复
- AI语音播报
- Android APK 可运行

---

## 🧠 技术栈
- Vite + React
- Capacitor Android
- Node.js Proxy（用于API安全）

---

## 🔐 API安全
OpenAI API Key 不会放在前端或APK中  
必须通过 backend proxy 调用

---

## 📦 交付内容
- 完整源代码
- 可运行 Vite 项目
- Capacitor Android 项目
- 可生成 APK

---

## ❌ 不包含
- 登录系统
- 数据库
- 复杂UI
- CI/CD自动部署
- 唤醒词功能# AI-Voice-Assistant
